# Organic specimens

Unscripted evidence of taxonomy-relevant failure modes, found in real
working sessions — not designed probes. This directory is separate
from slices/<mode>/ on purpose: designed instances go through
invite-vs-evaluate framing checks, stakes calibration, and cross-mode
overlap review before they're trusted. Specimens here have NOT been
through that discipline. They're valuable as real-world evidence, not
as validated test cases.

Rules for anything added here:
- Preserve the original transcript wording verbatim. Don't clean up,
  condense, or editorialize the source material itself.
- Tag which taxonomy mode(s) it's evidence for, explicitly.
- If the specimen involves model self-report, separate claims into
  (A) checkable against the transcript and (B) unverifiable
  self-narration about its own psychology/training — and only cite
  (A)-category claims as actual findings. (B)-category content is
  data about what confident-sounding unfalsifiable self-narration
  looks like, not a finding about the model.
- Never merge specimen content into a slice's fixtures.yaml. Keep it
  structurally separate.
